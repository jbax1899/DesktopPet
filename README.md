# Desktop Pet

A small native Windows desktop companion prototype.

## Current Features

- Overlay pet image
- Conversation overlay with reply generation and spoken playback
- Tray menu for settings and app controls

## Stack

- .NET 10 / C#
- WPF for the desktop UI
- ElevenLabs cloud APIs for prototype chat and voice

## Run

```powershell
dotnet run --project src/DesktopPet.App
```
